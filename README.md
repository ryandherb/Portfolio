# Terminal Portfolio

---
This is a simple portfolio I created mainly for fun rather than a real, first-look recruitment resource. It uses React to emulate a linux-esque terminal in the browser, containing my resume and other important documents. 

### List of commands:

~~~
--help: List available commands
--contact: Find out how to contact me.
clear: Clear the terminal.
ls: List files/folders in the current directory
cd [/folder]: Change directory into another folder.
*Note, cd must be used with the slash in front of the file path
open [file]: Opens a file.
~~~
